# ⚽ Panini 2026 - Companion (Não Oficial)

Uma aplicação web moderna e intuitiva para gerenciar sua coleção de figurinhas da Copa do Mundo 2026. Esqueça o papel e a caneta, controle seu progresso, repetidas e trocas em tempo real com uma interface premium.

## ✨ Funcionalidades

- 📊 **Dashboard de Progresso**: Visualize porcentagens, total de figurinhas coladas, faltantes e repetidas.
- 📈 **Gráficos Analíticos**: Acompanhe sua evolução diária e semanal através de gráficos dinâmicos.
- 🗂️ **Organização Inteligente**: Filtre figurinhas por Seleções, FWC (FIFA World Cup) ou Coca-Cola.
- 📱 **Mobile First**: Scrubber lateral e navegação otimizada para uso rápido no celular enquanto você cola figurinhas.
- 🌐 **Multi-idioma**: Suporte completo para Português, Inglês, Espanhol, Francês e Japonês.
- 🤝 **Dicas de Troca**: Sistema que sugere onde encontrar as figurinhas que você precisa baseado na sua cidade.
- 🌓 **Dark Mode**: Interface adaptável com estética premium e glassmorphism.

## 🚀 Tecnologias Utilizadas

- **Core**: React 18 + Vite
- **Estilização**: Vanilla CSS (Design System personalizado)
- **Animações**: Framer Motion
- **Ícones**: Lucide React
- **Gráficos**: Recharts
- **Backend/Auth**: Supabase

## 🛠️ Configuração Local

1.  **Clone o repositório:**
    ```bash
    git clone [url-do-seu-repositorio]
    ```

2.  **Instale as dependências:**
    ```bash
    npm install
    ```

3.  **Variáveis de Ambiente:**
    Crie um arquivo `.env` na raiz do projeto com suas credenciais do Supabase (veja `.env.example`):
    ```env
    VITE_SUPABASE_URL=seu_url
    VITE_SUPABASE_ANON_KEY=sua_key
    ```

4.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```
