# Geographical Insights

Purpose: Gerar estatísticas regionais anonimizadas sobre a disponibilidade de figurinhas para facilitar trocas.

## Requirements

### Requirement: Location Selection
Coleta opcional de localização para estatísticas.

#### Scenario: Onboarding Location
- **GIVEN** que é o primeiro login do usuário
- **THEN** o sistema deve solicitar opcionalmente a Cidade e o Estado.
