# Delta: FIFA 2026 Sticker Tracker

## MODIFIED Requirements

### Requirement: Dashboard Content
The dashboard (Home tab) is redefined as an introductory and informational view about the application's purpose.

#### Scenario: New Home View
- **WHEN** the user is on the "Home" (Início) tab
- **THEN** the system must display an introduction about the "Companion" app.
- **AND** explain the core goals: managing the collection, tracking progress, and finding swaps in the same city.
- **AND** statistics cards/progress charts should NOT be visible on this screen (moved to Progress tab).

### Requirement: Detailed User Statistics
#### Scenario: Location of statistics
- **WHEN** the user wants to see detailed metrics
- **THEN** they must navigate to the "Progress" (Progresso) tab.
