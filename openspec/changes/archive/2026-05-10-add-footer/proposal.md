## Why

Personalize the application with creator credits and provide a quick link to the developer's GitHub profile. This adds professional polish and acknowledges the work of Lucas Rogerio.

## What Changes

A new Footer component will be added to the bottom of the application. It will contain the text "Created by Lucas Rogerio" with a link to his GitHub profile. The footer will be responsive and integrated into the global layout.

## Capabilities

### New Capabilities
- `footer`: A global footer component for credits and external links.

### Modified Capabilities
- `app`: Update the main layout to include the Footer component.

## Impact

- `src/components/Footer.jsx`: New component.
- `src/App.jsx`: Main layout modification.
- `src/locales/*.json`: New localization keys for footer text.
